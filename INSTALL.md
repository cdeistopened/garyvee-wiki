# Installing Ask GaryVee

## Prerequisites

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) installed and working

## Installation

```bash
claude install-plugin github:cdeistopened/garyvee-wiki
```

Or add the plugin directory manually:

```bash
# From your project directory
claude plugin add /path/to/garyvee-wiki/plugin
```

## First Question to Try

```
Audit my last 15 posts
```

This routes to the 15-Post Diagnostic — GaryVee's flagship content audit — and gives you a complete diagnosis of your jab/hook ratio, selfless/selfish balance, platform-native adaptation, and hook quality.

## Other Good Starting Questions

- "Which platforms should I focus on?"
- "How do I get 30 pieces of content from one video?"
- "Help me with the $1.80 strategy"
- "Am I selling too much on social?"
- "Why aren't my posts getting views?"

## Playbook Sequences

For a full build, run skills in playbook order:

1. **Content Pyramid** (5 skills) — Turn one piece into 30+ platform-native pieces
2. **Day Trading Attention** (5 skills) — Find underpriced attention, amplify winners
3. **$1.80 Strategy** (3 skills) — Build audience through strategic engagement
4. **Jab-Right-Hook** (3 skills) — Balance giving and selling
5. **Content Audit** (3 skills) — Full diagnostic when things aren't working

## Troubleshooting

- **Skills not showing:** Make sure the plugin path is correct and Claude Code has been restarted
- **Generic answers:** Try invoking a specific skill directly: `/ask-garyvee:fifteen-post-diagnostic`
- **Not sure where to start?** The 15-Post Diagnostic is always the best entry point
